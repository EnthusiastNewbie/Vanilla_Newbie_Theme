## Come iniziare

Per provare uno dei temi, clona l'intero repository e poi entra nella cartella dedicata alla versione che preferisci:

```bash
# 1. Clona l'hub dei temi
git clone https://github.com/EnthusiastNewbie/Vanilla_Newbie_Theme.git

# 2. Entra nella cartella del repository
cd Vanilla_Newbie_Theme

# 3. Scegli la tua versione (Esempio: Hyprland)
cd Vanilla_Newbie_Theme_Hyprland

# 4. Una volta dentro la cartella specifica, segui le istruzioni del README locale per l'installazione.

```
